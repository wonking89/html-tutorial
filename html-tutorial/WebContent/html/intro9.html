1.html 주석 태그

다음과 같은 구문을 사용하여 html 소스에 주석을 추가할 수 있다.
  1. <!-- write your comments here -->
주석은 브라우저에 표시되지는 않지만 , html 코드를 문서화하는데 도움을 준다.
주석은 통해 html코드를 설명할수 있고, 문서를 강조하고 가독성을 높일 수 있으며,디버깅에도 도움을 준다
  1. <!-- this is a comment-->
  3. <p>This is a paragraph./p>
  5. <!-- Remember to add more information here here -->
※ Internet Explorer에서는 버전별 호환성 때문에 다음과 같은 조건부 주석의 사용도 가능하다.
  1. <!--[IT IE 5]>This is IE5 <br><![endit]>-->
  2. <!--[IT IE 6]>This is IE6 <br><![endit]>-->
  3. <!--[IT IE 7]>This is IE7 <br><![endit]>-->
  4. <!--[IT IE 8]>This is IE8 <br><![endit]>-->
  5. <!--[IT IE 9]>This is IE9 <br><![endit]>-->
