그러나, HTML을 배우기 위해 꼭 전문툴을 사용할 필요는 없습니다. PC에 설치된 간단한 텍스트 편집기를 사용하여 HTML을 배울 수있습니다.

2.윈도우즈의 메모장을 이용하여 html 작성

① 메모장(notepad)실행
② 메모장 html코드를 실행
③ html을 저장.(htm 또는 .html 확장자를 사용하면 딘다)
④ 웹 브라우즈에서 html파일 열기
