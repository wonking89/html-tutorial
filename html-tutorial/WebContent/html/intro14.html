1. table

  • 테이블은 <table>태그를 사용
  • 테이블 행은 <tr>을 사용
  • 행을 셀(데이터를 가지는 요소)로 분할 하는 태그는 <td>를 사용
  • 테이블의 너비, 테두리, 패딩, 셀간격, 경계설정 등은 CSS를 사용
