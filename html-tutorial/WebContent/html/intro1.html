1. html 예

  1. <!DOCTYPE html>
  2. <html>
  3. <body>
  4.     <h1>My First Heading</h1>
  5.     <p>My first paragraph.</p>
  6. </body>
  7. </html>
설명
  • DOCTYPE 선언은 문서의 형식을 정의 : <! DOCTYPE html> 은 이문서가 html5양식 문서임을 의미한다
  • <HTML>와 </ HTML> 사이의 텍스트가 웹 페이지를 설명한다
  • <BODY>와 </ BODY> 사이의 텍스트가 표시되는 페이지 내용입니다
  • <H1>와 </ H1> 사이의 텍스트는 제목을 표현한다
  • <P>와 </ P> 사이의 텍스트는 문단(단락)을 표현한다

2. html 이란?

HTML 웹 페이지를 기술하는 언어이다.
  • HTML은 Hyper Text Markup Language의 줄임말
  • HTML은 (는) 마크 업 언어이다
  • 마크 업 언어는 태그의 집합이다
  • 태그 문서의 내용을 설명한다
  • HTML 문서는 HTML의 태그 및 일반 텍스트를 포함한다
  • HTML 문서는 웹 페이지라 부른다

3. html 태그

HTML 마크 업 태그는 일반적으로 HTML 태그라고합니다
  • HTML 태그는 꺽쇠괄호로 둘러싸인 태그이름을 의미한다 ex)<html>
  • HTML 태그는 보통 시작 태그(<b>)와 종료 태그(</b>)의 쌍으로 이루어 진다
  1. <tagname>content</tagname>

4. html 요소

HTML 태그와 "HTML 요소는 같은 의미로도 사용되지만, 엄격히 말하면 HTML 요소는 시작 태그와 종료 태그 사이의 모든것을 의미한다. (html요소안에 html태그가 포함되어 있을 수도 있다)
  1. <p>
  2. <div>content</div>
  3. </p>

5. 웹 브라우저

웹 브라우저(구글 크롬, 인터넷 익스플로러, 파이어 폭스, 사파리 등) 의 목적은 HTML 문서를 읽고 웹 페이지로 표현(rendering)하는 것입니다. 브라우저는 HTML 태그를 해석하여 HTML 페이지의 콘텐츠를 사용자에게 표현한다.

8. <!DOCTYPE> 선언

<! DOCTYPE> 선언은 브라우저가 웹페이지를 어떤 형식(문법)으로 읽어야 할지를 결정하는데 도움을 준다
이 웹상에는 많은 다른 버전의 html문서가 있지만, 웹 브라우저는 웹페이지가 어떤 형식과 버전으로 만들어 졌는지만 알면 정확히 표현할 수 있다.
  1. // HTML5
  2. <!DOCTYPE html>
  4. // HTML 4.01
  6. <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
  8. "http://www.w3.org/TR/html4/loose.dtd">
  11. // XHTML 1.0
  12. <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
  13. "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
